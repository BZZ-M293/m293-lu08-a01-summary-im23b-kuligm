Zusammenfassung Modul 293

Block-Elemente Div-Element p-Element Table-Element

Elemente

Block-Elemente

Mit section kann ein Abteil definiert werden.

Mit den h1-6 tags kann man Überschriften schreiben. Dabei ist h1 die wichtigste (und grösste) Überschrift und h6 die unwichtigste

Mit article kann ein Text definiert werden. Darin sind dan Paragrafen mit dem "p" Tag.

Div-Element

Der div-Tag wird als container verwendet. Jeder Inhalt kann als div-Tag eingefügt werden. Er kann auch mit css gestaltet werden.

p-Element

Mit dem p-Tag kann man Texte schreiben. Pro absatz wird ein P-Tag benutzt.

Table-Element

Mit dem table-Tag wird eine Tabelle erstellt. Weitere Tags können dann weitere Elemente in der Tabelle hinzufügen:

  • tr: table row
  • th: table header
  • td: normale zelle
